## Overview
This project demonstrates feature extraction from images using
ORB (Oriented FAST and Rotated BRIEF) and SIFT (Scale-Invariant Feature Transform).

## Tools & Libraries
- Python
- OpenCV (opencv-contrib-python)
- NumPy
- Pandas
- Matplotlib
- skimage

## Dataset
Images are taken from skimage built-in datasets.
A CSV file (image_dataset.csv) is generated containing image names.

## Steps
1. Install required libraries
2. Load grayscale images
3. Create CSV dataset
4. Initialize ORB & SIFT
5. Extract keypoints
6. Visualize keypoints
7. Compare ORB vs SIFT using graphs

## Output
- Keypoint visualizations
- ORB vs SIFT comparison graph
- Downloadable CSV file

## Conclusion
SIFT generally detects more keypoints and is more robust,
while ORB is faster and suitable for real-time applications.
