## PROJECT OVERVIEW
This project demonstrates a simple Perceptron model, which is the foundational
building block of neural networks. The model performs binary classification on
a linearly separable dataset.

## OBJECTIVES
- Understand how a Perceptron works
- Train a Perceptron using gradient updates
- Visualize training error
- Plot decision boundary

## DATASET DETAILS
File: perceptron_data.csv

Columns:
- x1 : Feature 1
- x2 : Feature 2
- label : Target class (0 or 1)

## TECHNOLOGIES USED
- Python
- NumPy
- Pandas
- Matplotlib
- Google Colab

## MODEL DESCRIPTION
The Perceptron algorithm updates weights using:
    w = w + learning_rate * (target - prediction) * x

Activation Function:
- Unit Step Function

## STEPS TO RUN
1. Open Google Colab
2. Upload perceptron_data.csv
3. Copy and paste the provided Python code
4. Run all cells

## OUTPUTS
- Training error vs epochs graph
- Decision boundary visualization
- Learned weights and bias

## LEARNING OUTCOME
This project builds intuition for:
- Linear classification
- Neural network fundamentals
- Model convergence behavior
